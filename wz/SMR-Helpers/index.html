“SMR Helpers”面板 （SMR）

此面板是 Smart Magnet Rig 工作流程的一部分

功能性

灵活、可定制的面板，带有多个按钮，可轻松访问与智能_磁铁绑定_的绑定、编辑、姿势、作或修复相关的功能。

Premium 许可证也可以通过 SMR Helpers 面板进行激活和管理。

建议的使用方式

涉及 Smart Magnet Rigs 的各种作。请参阅下面的完整列表。

它在哪里工作？

大多数作都需要 Stage 或 Timeline 选择

EDAP 工具快捷方式

不适用;从“窗口”菜单打开_“SMR 帮助对象_”>其他面板（Animate 中的扩展）> SMR 帮助对象

定制、交互

面板可以作为水平或垂直条带或标准矩形面板停靠。

按住 Shift 键并拖动按钮可以更改其位置。
按住 Shift 键并拖放到面板边界之外的按钮可隐藏该按钮。
可以通过将按钮从右下角的 Separator Pool [ | ] 中拖动来添加按钮之间的分隔符。

点击右下角的 [ ··· ] 会弹出_设置对话框_，其中的复选框允许隐藏/显示每个按钮。
它还提供了几种预设组合。

按钮可以显示在带有文本标签的展开视图中，也可以以仅图标的紧凑视图显示。

按钮的完整列表

IK Effector* – 创建 SMR IK 效应器。阅读更多…

IK Brace* – 创建 SMR IK Brace。阅读更多…

Show / Hide Controllers* – 切换SMR_Controllers层的可见性。阅读更多…

Controllers Opacity* – 将所有 SMR 控制器的不透明度逐渐降低 20%。阅读更多…

Move / Solve* （移动/解算）* – 根据初始选择调用 Move （移动） 或 Solve IK 链作。阅读更多…

Effectors State* （效应器状态） – 将附加的 SMR 效应器的状态切换为 RIK Active （RIK 活动） 或 RIK Inactive （RIK 非活动）。阅读更多…

Effector Presets* – 显示 Effector Presets Manager 对话框。预设保存在 SMR 帮助程序面板本身中，并且可以在各个文档中使用。

Effector Properties* （执行器属性） – 显示当前所选 SMR 效应器的 Effector Properties （执行器属性） 对话框。

Center Marker （中心标记） – 创建中心标记。阅读更多…

Magnet Target （磁铁目标） – 创建 Magnet （磁铁） 目标。阅读更多…

FK Extender* — 创建 FK Extender。阅读更多…

Move / Rotate* – 创建 Move / Rotate Transformer。阅读更多…

Scale Horizontally* （水平缩放） – 创建 Horizontal Scale / Skew Transformer（水平缩放/倾斜变换器）。阅读更多…

Scale Vertically* （垂直缩放） – 创建 Vertical Scale / Skew Transformer（垂直缩放/倾斜变换器）。阅读更多…

T 点* – 创建 T 点。阅读更多…

Pose Anchor* （姿势锚点） – 创建 Pose Anchor（姿势锚点）。阅读更多…

RC Sliders – 弹出一个用于创建 RC 滑块的对话框。阅读更多…

拉直控制器 – 如有必要，水平翻转 Transformer 和 RC 滑块，以使用户文本标签易于阅读。阅读更多…

SMR Sample （SMR 示例） – 打开一个示例文件，用户可以在其中测试 Inverse Kinematics 并探索其他 Smart Magnet Rig 功能和工具。

TP Sample* – 打开一个示例文件，用户可以在其中测试 Transformers、RC Sliders 和 TransPoser，以及 Smart Magnet Rig Forward 和 Inverse Kinematics。

Change Master （更改主数据库） – 调用 Change Master Parent （更改主数据库父级） 脚本。必须在舞台上选择 Smart Magnet Rig 的一个元素。它将成为新的主父级。阅读更多…

SMR Break Apart （SMR 分离） – 分离一个元件，其中包含一个 Smart Magnet Rig。阅读更多…

SMRize Parented （SMRize Parented） – 将 Parented Layers 的现有层次结构转换为 Smart Magnet Rig。初始选择必须包括 PL 层次结构的所有元件，无论是在舞台上还是在时间轴上。阅读更多…

IK Clean Up* – 清理传统补间，通过为补间元素的每一帧创建关键帧来解决 IK 链问题。初始选择必须是一个 SMR IK 效应器。阅读更多…

KineFlex 变换元件焊接机 – 将智能磁铁装置的选定元件焊接到其父项上。需要在舞台上选择一个或多个元素。阅读更多…

Make Edaptable （使可编辑） – 匹配 Transformation （变换） 和 Registration （注册点） 的位置，以便可以将元素作为 Smart Magnet Rig 进行装配。阅读更多…

Roll Over Rig info （滚动绑定信息） – 将 Roll Over Rig 信息（SMR 元数据）移动到当前关键帧的左侧或右侧。需要 Keyframe 选择。阅读更多…

Edit M Targets （编辑 M 目标） – 启动 Inspect / Edit Magnet Targets （检查/编辑磁体目标） 对话框。必须选择 Magnet Target 进行检查。阅读更多…

Remove Rig info （删除 Rig 信息） – 从选定元素中删除 Smart Magnet Rig 元数据 （Rig 信息）。阅读更多…

KineFlex Switch* – 按住 [ Alt+Shift ] 修饰键时，在两种直接作 SMR 元素的 KineFlex 模式之间切换 – Frame Scrub 或 Squesh & Stretch。阅读更多…

License Manager – 显示 EDAPT License Manager。阅读更多…

[ ··· ]– 打开 SMR Helpers Settings 对话框。

笔记

所有 EDAPT 对象（如磁铁目标、中心标记、SMR 效应器等）都放置在名为 EDAPT 对象的“库”文件夹中。

请勿重命名、移动或删除此文件夹。它应该始终保留在 Library 的根目录中。

视频演示

我们所有的视频都是 1080p。如果由于某种原因此处无法提供全高清，请尝试在 YouTube 上观看。

① 点击“1点此安装.jsfl”；

② 选“作为命令运行”，点击“确定”；

③ 继续点“确定”;

④ 勾选“l accept …”，点击“确定”；

⑤ 点击“Quit”；

⑥ 重启AN，在“窗口->拓展”中打开“SMR helpers”使用

3卸载步骤

① 重启AN；

② 依次打开“窗口”->“拓展”->“EDAPT Commands”；

③ 下拉列表，点击“EDAPT Control Panel”;

④ 点击左下角“Uninstall EDAPT”；

⑤ 点击“确定”。

⑤ 点击“Quit”。
